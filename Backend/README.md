# AI Interview Coach

AI Interview Coach is a web application that helps users practice technical interviews using AI.  
The system generates interview questions and evaluates user answers with feedback and scoring.

## Tech Stack

Frontend
- React.js
- Tailwind CSS

Backend
- Python
- FastAPI

AI Integration
- OpenAI API
